Weathercast marche grâce à Electron. Il s'agit d'un environnement permettant de développer des applications bureau avec des technologies web

Electron a notamment permis de développer Discord, Spotify, Skype, Twitch, etc...

Les API utilisées dans Weathercast sont :

  • Nominatim
  • Open-meteo
  • Geonames
  • Timezonedb

Le CSS est géré par Tailwind.

Tailwind CSS est un framework CSS qui fournit des classes CSS préconçues pour faciliter la création d'un site web.

Oui le code de Weathercast est open source.

Il est consultable via le github associé ici.
